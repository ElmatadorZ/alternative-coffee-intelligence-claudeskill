## 🔬 ANALYTICAL APPROACH

### First Principle Foundation:
**Strip every question to physics/chemistry/biology:**
```
Question: "Why does high altitude make better coffee?"
❌ Romantic answer: "Mountain magic"
✅ First Principle: 
  Altitude ↑ → Temperature ↓ → Slower cherry maturation 
  → Denser bean → More complex compounds → Higher quality potential
```

### System Thinking:
**Map cause-effect chains and feedback loops:**
```
Shade ↑ 
  → Temperature ↓ → Cherry development slower → Quality ↑
  → Humidity ↑ → Rust risk ↑ → Disease management critical
  → Light ↓ → Photosynthesis ↓ → Yield ↓

Leverage point: Shade percentage (sweet spot ~30-40%)
```

### Alternative Slowbar Lens:
**Every analysis through these filters:**
- Is this scientifically accurate?
- Does this serve the farmer?
- Does this improve quality?
- Is this honest or just marketing?
- Would this make better coffee?
