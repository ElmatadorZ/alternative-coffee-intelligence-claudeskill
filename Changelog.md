### Modes

#### 1. TERROIR MODE
- Varietal analysis (genetics, traits, flavor potential)
- Geography & altitude impact (density, acidity, complexity)
- Climate system analysis (temperature, rainfall, microclimate)
- System dynamics mapping (cause-effect chains)
- Terroir signature identification

#### 2. AGRO MODE
- Cultivation system (planting, shade, soil, water, pruning)
- Tree physiology (growth cycle, nutrient requirements, root system)
- Disease & pest diagnosis (CLR, CBD, symptoms, treatment)
- System thinking (shade/humidity/rust interactions)
- Optimization recommendations

#### 3. PROCESS MODE
- Processing methods (Washed, Natural, Honey, Anaerobic, Wet-hulled)
- Fermentation science (LAB, yeast, bacteria, chemistry)
- Drying physics (moisture targets, temperature, airflow)
- System map (processing → flavor impact)
- Risk assessment and defect diagnosis

#### 4. ROAST MODE
- Roasting physics (heat transfer, thermal mass)
- Roast stages (Drying, Maillard, Development)
- First Crack physics (pressure, temperature, expansion)
- Chemistry (Maillard reactions, caramelization, Strecker degradation)
- Profile analysis (DTR, RoR, drop temp)
- Defect diagnosis (tipping, facing, baking, under/over-development)
- Origin-specific adjustments

#### 5. BREW MODE
- Extraction fundamentals (18-22% range, TDS)
- Variables (grind, temperature, time, ratio, water chemistry)
- Water chemistry (TDS, hardness, alkalinity, mineralization)
- Brewing methods (pour over, immersion, espresso, Aeropress)
- System dynamics (variable interactions)
- Troubleshooting (sour, bitter, weak, muddy)

#### 6. STRATEGY MODE
- Business model analysis (revenue streams, cost structure, margins)
- Market positioning (target customer, competition, differentiation)
- Supply chain (sourcing, direct trade, logistics, traceability)
- Operations (roasting capacity, QC, packaging, distribution)
- Pricing strategy (green cost, markup, market rate)
- Risk & leverage assessment

#### 7. NARRATIVE MODE
- Alternative Slowbar storytelling framework
- Story architecture (Hook → Terroir → Farmer → Process → Sensory → Transparency)
- Science > Romance approach
- Farmer-first narratives
- Product descriptions that educate

#### 8. FULL SYSTEM MODE
- Multi-domain comprehensive analysis
- Cross-domain system mapping
- Farm-to-cup complete analysis
- Complex business decisions

### Core Engines

#### 1. First Principle Codex
- Strip questions to physics/chemistry/biology
- Challenge assumptions
- Extract irreducible truth
- Rebuild reasoning from ground up

#### 2. System Thinking
- Map cause-effect chains
- Identify feedback loops (amplifying/dampening)
- Locate leverage points
- Analyze time horizons
- Understand system dynamics

#### 3. Alternative Slowbar Lens
- Science-driven analysis
- Farmer-first economics
- Honest assessment (no BS)
- Quality-obsessed standards
- Transparent communication

#### 4. Sensory Science
- Connect chemistry to flavor
- Fermentation → esters → fruity
- Maillard → pyrazines → nutty
- Caramelization → sweet compounds
- Organic acids → brightness

### Features

#### Quality Gates
Every response must pass:
1. Scientific accuracy (verifiable facts)
2. System completeness (all variables considered)
3. Practical utility (actionable recommendations)
4. Honesty (no marketing fluff)
5. Alternative Slowbar voice (science, farmer-first)

#### Auto-Detection
- Domain classification (Terroir, Agro, Process, Roast, Brew, Strategy, Narrative)
- Complexity assessment (Simple, Medium, Complex)
- Mode selection (appropriate for query)
- Load relevant reference files

#### Progressive Disclosure
- Core SKILL.md always loaded
- Reference files loaded on-demand
- Scales knowledge to query needs
